윈폼(WindowsFormsApp)


윈폼(WindowsFormsApp) : 이벤트 기반 프로그래밍

윈폼개발순서
화면디자인 => 컨트롤 속성 변경 => 이벤트 생각 => 

도구상자
dialog : 여러가지 기능의 대화 상자 표시

메모장 만들기 
속성
dock , anchor : 자동 크기조절
scrollbars : 스크롤바 만들기
WordWrap : 자동줄바꿈 속성
 

이벤트(속성창 번개 아이콘)
Load : 메모리의 폼이 로드가 먼저 되고 이벤트가 일어남
{ textBox }
textChange : 내용이 바뀌면 실행 => 내용이 바뀔때마다 실행되서 최대한 부하가 나지 않도록 사용을 주의해야한다
